CSS Cheatsheet

Selectors

Selector Name Description
<.class> class name using the element's class name
<#id-name> element id uses the element id
<.class> class uses the element class name
<h1, p> multiple tags uses multiple tag names
<h1, .class> mixture mixture of selectors
<h1.class> chaining selectors selects h1 elements, with class of "class"
<.main-list li> nested elements nested "li" within .main-list
